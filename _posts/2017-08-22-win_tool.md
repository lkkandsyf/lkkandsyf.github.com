## window shell

[babun](http://babun.github.io/),Babun - a window shell you love,解压出来，直接运行bat，等待安装，之后就可以使用了，有find,grep,awk,sed等.

[configure](http://www.rxna.cn/post/wiki/babun-pei-zhi)

+ babun中文乱码
add code to ~/.babun.rc
```
chcp 65001
:wq
source ~/.babunrc
```
