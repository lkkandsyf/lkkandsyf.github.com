## 安装(Ubuntu ntop)

	sudo apt-get install ntop

	sudo /etc/init.d/ntop start
	sudo net -tulpn | grep 3000		// 输出监听的端口
	sudo apt-get autoremove ntop	// 只下载这个软件

这里需要创建一些东西，例如，端口，密码，端口默认是3000.密码自己设置。

打开浏览器，输入localhost:3000
