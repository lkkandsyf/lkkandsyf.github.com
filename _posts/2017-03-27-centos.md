## **安装完成之后，需要哪些事**

### sudo problem

open terminal

{% highlight C linenos %}
# change to root
visudo #open sudos
# add username by the root
root ALL=(ALL) ALL
username ALL=(ALL) ALL
:wq
{% endhighlight %}

### network problem

```C
# change to root
cd /etc/sysconfig/network-scripts/
# ls and find network card
vim ifcfg-xxxx	#edit
# ONBOOT=no change to ONBOOT=yes
:wq

# restart network
service network restart
ping baidu.com
```
